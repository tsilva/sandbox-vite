# sandbox-vite

A sandbox project for experimenting with Vite - the next generation frontend tooling.

## About

This project serves as a playground for exploring Vite's features, including:
- Lightning fast HMR (Hot Module Replacement)
- Out-of-the-box TypeScript support
- Modern development environment

## Getting Started

1. Clone the repository
2. Install dependencies:
```bash
npm install
```
3. Start development server:
```bash
npm run dev
```
